# 📈 Full-Stack Stock Trading Platform

A full-stack stock trading web application inspired by modern trading platforms like Zerodha. This platform allows users to track portfolios, manage orders, and visualize stock data through interactive charts.

---

## 🚀 Features

- 📊 Real-time stock dashboard UI
- 💼 Portfolio management (Holdings & Positions)
- 🛒 Buy/Sell stock functionality
- 📈 Interactive charts and graphs
- 🔐 Secure backend with REST APIs
- ⚡ Responsive and user-friendly interface

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- HTML5, CSS3, JavaScript

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Other Tools:**
- Chart.js
- Git & GitHub
